Codex Feishu Bridge for macOS

1. Double-click "Install Codex Feishu Bridge.command".
2. Enter the Feishu App ID and App Secret in Terminal.
3. Send the displayed /pair command to the Feishu bot.

If macOS blocks the first launch because this community package is not signed,
Control-click the installer, choose Open, and confirm once.

Requirements:
- macOS 13 or newer
- Node.js 22 or newer
- The codex command available in Terminal

Credentials are stored in the current user's macOS Keychain. Runtime data is
stored in ~/Library/Application Support/CodexFeishuBridge.
